# CS326_Automation_Tool

Αυτό είναι ένα αυτοματοποιημένο εργαλείο ανάλυσης και εκμετάλλευσης ευπαθειών μνήμης, ανεπτυγμένο ειδικά για τις ανάγκες του τελευταίου quiz του μαθήματος **CS326**.

## 📖 Περιγραφή
Το **CS326_Automation_Tool** επιτρέπει την αυτόματη αναγνώριση των προστασιών ενός binary και τη δημιουργία του κατάλληλου payload (ROP ή Shellcode) για την απόκτηση κελύφους (shell). Το εργαλείο εστιάζει στην ακρίβεια των offsets και στην παράκαμψη του **NX-bit (No-Execute)**.

## 🚀 Οδηγίες Χρήσης

Για να τρέξετε το εργαλείο, χρησιμοποιήστε την παρακάτω εντολή:

```bash
printf '#! /bin/sh\nenv -i TEMP=1000 setarch i686 -R -3 "$@"\n' > run.sh
chmod 777 *
python3 auto_solver.py bin.1
